March 21, 2025

Dashboard Scrub, New Sources, Admin Settings & More

✦ Shipped

  • Dashboard Scrub

    Completed a full review of the Dashboard. Removed any metrics that weren't calculating correctly — we'll revisit those once we have the ability to set a Source on a Sale, which will resolve the underlying logic issues. Tooltips have been added anywhere that was unclear or potentially confusing.

  • New Sources & LOB

    Added new Sources and a new Line of Business to support the data migration:

    • New Organic
    • New Networking
    • New Acquired Book of Business
    • New Referral – Next Gen Assets
    • New Annual Review
    • Updated Partnership → renamed to Strategic Alliance
    • New LOB Estate/Trust
  • Admin Settings (Beta)

    Built an initial Beta version of the Settings Page. From here you can configure LOB Revenue Calc percentages, and create or edit Group Members. You can also create a new group — Liz can walk through those steps. Note: Currently only visible to internal ARC users via impersonation. Not visible to Advisors.

  • Advisor Selector

    Added an Advisor Selector to both the Opportunity Create modal and the New Event modal. This allows the logged-in user to select a different Advisor inline if needed, rather than relying solely on the preset.

  • Estimated Cost on New Event Modal

    Estimated Cost field has been added directly to the New Event modal.

🚀 Roll Out Update

  • Full Access Starting Monday

    Starting Monday, everyone who had access to Old Tracker will have access to New Tracker. If you notice any issues, please let us know right away.

  • Bugs vs. Feedback

    As Advisors start getting in and training begins — please continue to flag any TRUE bugs immediately. For general feedback, we'll keep that to email going forward. We'll continue prioritizing the build out and making ongoing improvements to Tracker!

⟳ In Progress

  • Duplicate Contact Detection & Merge

    Working on a button that will identify potential duplicate contacts and allow you to merge them. This will be especially useful post-migration for cleaning up any dupes. Targeting release early next week.

March 13, 2025 Feature Enhancement
Dashboard Overhaul, New Event Tile & Opportunities Improvements

A big week — lots of highly-requested updates across the Dashboard and Opportunities screen, plus a brand-new Event tile:

  • New Event Tile on the Dashboard — A highly requested addition! All Events and their key data points are now laid out in one comparison table, complete with individual figures and totals. Big win based on your feedback.
  • Sort by Advisor on Opportunities Screen — You can now sort the Opportunities list by Advisor directly from the screen.
  • Edit Source on Tracked Events — Previously locked down, you can now edit an opportunity source even when that source is a tracked Event.
  • Additional Dashboard Tool Tips — Added more tooltips in areas where we've received clarifying questions, to reduce ambiguity right in the product.
  • Prior Year Date Filters on Dashboard — The Dashboard date filter now includes prior years (2023, 2022, etc.) when data exists for those periods.
  • Opportunities Date Filter Redesign — The date filter on the Opportunities screen is now a dropdown that matches the Dashboard's date filter, keeping the experience consistent and simpler.
  • Costs & ROI Added to Dashboard Export — Cost breakdown data is now included in the Dashboard Export for an initial view of ROI.
March 6, 2025 Feature Enhancement
Dashboard Data, Opportunity List View & Advisor Reassignment

A busy week of updates based on advisor feedback — here's everything that changed:

  • Opportunity Created Date in List View — Now visible in the List View. Sort by this column to bring the newest opportunities to the top. Filter using the new Sale Date and Created Date Range options.
  • Advisor Reassignment — Change the Advisor directly from the Opportunities and Events screens. When selecting a new Advisor you'll be prompted to reassign any existing sales too. More improvements planned — but this should be a big lift right now.
  • Production & Revenue tile — A Total column has been added showing overall numbers alongside the category breakdown.
  • ROI Analysis tile — A Totals row has been added at the bottom of the table.
  • Sales & Opportunities dashboard — A Totals row added for at-a-glance summary figures.
  • Paid with Marketing Dollars — A checkmark for Paid with Marketing Dollars has been added to the Event Create modal.
